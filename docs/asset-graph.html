Asset Graph

Asset Graph is a powerful system designed to compose a searchable graph of files based on a set of soft and hard contextual associations. The system is architected as a modular framework with extension points for expanding supported file types and relational mapping techniques, making it highly flexible and customizable to the dataset and application.

With Asset Graph, you can facilitate the ingestion and enrichment of large amounts of assets spanning over many years of production, as demonstrated by its successful implementation for a notable animated YouTube series. In this project, Asset Graph was used to manage over 400k+ assets, including images, videos, audio, and text files, and to organize and analyze the relationships between them. The datasets produced by this system are ideal candidates for producing training datasets for generative AI models.
